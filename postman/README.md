# KulmiPay Postman Collection

## Files

- `kulmipay-public-api.postman_collection.json` - API requests using live URLs.
- `kulmipay-sandbox.postman_environment.json` - optional sandbox variables.
- `kulmipay-live.postman_environment.json` - optional live variables.

## Import

1. Open Postman.
2. Import `kulmipay-public-api.postman_collection.json`.
3. Replace placeholder values such as `<secret_key>`, `<public_key>`, `<file_id>`, and `<tracking_id>`.

## Optional Variables

- `api_host`
- `base_url`
- `secret_key`
- `public_key`
- `phone_number`
- `api_ref`
- `wallet_id`
- `destination_wallet_id`
- `invoice_id`
- `tracking_id`
- `file_id`
- `transaction_id`
- `account_number`
- `bank_code`
- `callback_url`
- `chargeback_id`
- `paybill_account_id`
